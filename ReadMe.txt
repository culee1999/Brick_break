<Brick Break>

<실행 환경>
Window 11, Visual Studio 2017

<키보드, 마우스>
'S' or 's' : 게임 도중에 일시 정지 시킬 수 있음
마우스 왼쪽 클릭 : 스테이지 선택 할 때 사용됨
마우스 오른쪽 클릭 : 공을 발사시킬 때 사용됨

<게임 설명>
아래쪽에 위치한 '바'와 '공'을 이용해 화면에 있는 모든 벽돌을 깨는 것이 게임 목표이다.

<실행 방법>
게임을 실행시킨 후, 스페이스 바를 눌러 스테이지 선택 화면으로 넘어간다.
마우스 왼쪽 클릭으로 스테이지 선택 후, 마우스 오른쪽 클릭으로 벽돌깨기 게임을 진행한다.
모든 벽돌을 다 깨면 게임 클리어
모든 벽돌을 다 깨기 전에, 목숨 3개가 다 떨어지면 게임오버
